# Airbnb-Analytics-Dashboard
A comprehensive, end-to-end Power BI dashboard analyzing Airbnb dataset performance. Features interactive pages for Overall Overview, Price Analysis, Location Analysis, Property Analysis, and Rating &amp; Reviews to deliver insights on property type distributions, pricing trends, and customer metrics via dynamic KPIs, maps, and advanced visualizations.
